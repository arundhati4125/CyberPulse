# CyberPulse AI

## AI-Powered Network Intrusion Detection, Threat Severity Analysis & Response Recommendation System

CyberPulse AI is an intelligent cybersecurity platform that detects suspicious network traffic using Machine Learning, evaluates the severity of attacks, recommends immediate preventive actions, and provides AI-based explanations to help security analysts respond quickly.

### Features
- AI-Based Intrusion Detection
- Threat Severity Analysis
- Immediate Prevention Recommendations
- AI Security Assistant
- Incident History
- Interactive Dashboard
- Report Generation

Developed as a Engineering Project just out of curiosity.
